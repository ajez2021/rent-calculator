# Rent-to-Income Calculator 🏠

A simple, responsive rent affordability calculator that helps users estimate how much rent they can afford based on their annual income and a selected income multiplier (2.5x, 3x, 3.5x).

---

## ✨ Features

- Enter annual income
- Choose affordability multiplier:
  - 2.5x (Conservative)
  - 3x (Standard)
  - 3.5x (Aggressive)
- Instantly calculates:
  - Maximum annual rent
  - Maximum monthly rent
- Lightweight and fast
- Fully client-side (no backend required)
